Passo para criar um relacionamento 1 para N:

1. Criar as classes modelo (entity);
2. Anotar as classes com o Data JPA - criação das tabelas no BD;
3. Verificar no BD H2 a criação das tabelas e seus relacionamentos;
4. Criar a camada de repository;
5. Criar a camada de Dto;
6. Criar a camada de service;
7. Criar a camada de controller;
8. Testar as APIs no Postman;
9. Verificar os dados no console do H2.
